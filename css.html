CSS

CSS (Cascading Style Sheet) merupakan bahasa yang digunakan untuk mendeskirpsikan elemen HTML. Tugas HTML ada membuat elemen pada laman web, sedangkan tugas CSS adalah mengatur tampilan dari elemen tersebut. Contohnya: kita dapat membuat sebuah tombol dengan menggunakan tag HTML 'button', dengan CSS kita dapat mengatur warna bentuk, dan tampilan pada tombol tersebut.

CSS pertama di usulkan oleh Hakon Wium Lie dan Bert Bos tahun 1994 yang dimana mereka juga bekerja bersama Tim Berners-Lee (penggagas HTML). CSS diciptakan untuk memisahkan penulisan tampilan halaman dengan konten halaman sehingga memudahkan pengaturan saat menulis halaman web yang panjang atau kompleks.

Penulisan CSS

CSS mempunyai bentuk penulisan sepeti berikut

h1 {color: red;}

kode di atas mempunyai beberapa elemen yaitu; h1 -> sebagai selector, color -> sebagai property, dan red -> sebagai value

sebagai aturan umum, kita menulis elemen yang kita seleksi, kemudian dalam kurung, kita tulis properti yang akan kita ubah dengan nilai yang kita tentukan value-nya. misalnya sebuah teks kita rubah warnanya, gambar kita berikan bingkai, dsb

Penulisan CSS dapat dilakukan dengan tiga cara: inline, internal, dan eksternal.

  • Contoh inline: (dilakukan langsung pada elemen dengan atribut style) 

    sebuah heading

  • Contoh internal: (ditulis dalam tag 'head' menggunakan tag 'style')

    sebuah heading

Dan yang terakhir; eksternal (ditulis di dalam file lain dan di hubungkan dengan tag 'link')

contoh di atas, kita tuliskan tag link dalam tag head pada halaman html yang menunjuk pada file css, untuk penulisan css eksternal, kita tuliskan pengaturan css dalam file yang berbeda.

Selector CSS

Selector pada CSS juga mempunyai beberapa fitur. jika kita menuliskan tag html pada selector, maka seluruh elemen yang sama pada halaman tersebut akan berubah. Misalnya kita memberikan tag paragraf dengan warna font putih, maka seluruh paragraf akan berwara putih. Karena itu, untuk mengatur ini, kita dapat membuat selector sendiri menggunakan class (dengan tanda titik '.') atau id (tanda pagar '#')

Class Selector

kita dapat menciptakan selektor sendiri dan memberikan atribut class pada tag html. contoh:

.merah {color:red;}

paragraf berwarna merah

paragraf tidak merah

contoh di atas kita menulisan kode css dengan class '.merah' dan pada sebuah paragraf kita berikan atribut class dengan value merah, dan paragraf lain dengan class biru.

ID Selector

ID selector berfungsi sama seperti class selector, tetapi menggunakan tanda pagar (#), atribut id, dan hanya berlaku pada satu elemen pada halaman contoh:

#biru {color:blue;}

paragraf berwarna merah

paragraf menjadi biru karena id selector

contoh di atas kita menulisan kode css dengan id = #biru.
